初识ACM——这都是啥-.-

Dec 19, 2015
阅读时长: 1 分钟

学长讲解输出及数据类型 
代码是这样的：

#include <cstdio>
int main (){
    char c;
    char d;
    int a = 1234;
    float f = 3.141592653589;
    double x = 0.12345678987654321;
    c = '\x41';//十六进制 
    d = '\101';//八进制 
    printf("a = %d\n", a);
    printf("a = %6d\n", a);
    printf("a = %2d\n", a); 
    printf("a = %-6d--\n", a);
    printf("a = %06d\n", a);
    printf("a = %02d\n", a);
    printf("f = %f\n", f);
    printf("f = %6.4f\n", f);
    printf("x = %lf\n", x);
    printf("x = %18.16lf\n", x);
    printf("c = %c\n", c);
    printf("c = %d\n", c);
    printf("c = %x\n", c);
    printf("d = %c\n", d);
    printf("d = %d\n", d);
    printf("d = %x\n", d);
    return 0;
}

我的内心是崩溃的-.- 
但现在看看其实还好 也是开始向C语言探索迈步
