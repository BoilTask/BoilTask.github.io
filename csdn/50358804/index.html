初识ACM——渐入佳境

Dec 19, 2015
阅读时长: 1 分钟

渐渐的…… 
我也能写出来一些当时看很复杂的代码了…… 
这是一个奇偶数分离的程序

#include<stdio.h>
int main()
{
    int i,k,n;
    scanf("%d",&i);
    int a[i] ;
    for(k=0;k<i;k++)
        {
        scanf("%d",&a[k]);
        }
    for(k=0;k<i;k++)
        {
            for(n=1;n<=a[k];n++)
                {
                if(n%2==1)
                printf("%d ",n);
                }
            printf("\n");
            for(n=1;n<=a[k];n++)
                {
                if(n%2==0)
                printf("%d ",n);
                }
                printf("\n\n");
        }   

    return 0;
}

虽然现在看能改进的地方挺多的 
但是…当时看…已经很不错了好吧 ！
